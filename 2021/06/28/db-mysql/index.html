    • 地址：https://dev.mysql.com/downloads/

  • 安装：

    • 产品选择：

      • Developer Default：开发者默认，安装 MySQL 开发所需的所有产品。
      • Server only：服务器，只安装 MySQL 服务器产品。
      • Client only：客户端，只安装没有服务器的 MySQL 客户端产品。
      • Full：完全，安装所有包含的 MySQL 产品和功能。
      • Custom：手动，手动选择系统上应安装的产品。
